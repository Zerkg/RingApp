Программа может "браковать" некоторые аудиофайлы. Поэтому сразу после того, как будет выбран 
аудиофайл, нужно будет проверить его длительность в конфигурации звонка. Если она будет равна нулю, 
то программа НЕ БУДЕТ воспроизводить этот файл. Лечится это тремя способами - либо найти другой файл, 
либо перерендерить существующий (например, тут https://online-audio-converter.com/ru/), либо с
помощью стороннего редактора тэгов (НЕ СИСТЕМНОГО) удалить тэг ID3 v2 (например, с помощью AIMP).
